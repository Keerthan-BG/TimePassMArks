## 📖 About The Project

🔐 **Digital Twin Cyber Security in IIoT** is an intelligent monitoring system that creates a virtual replica of an industrial process and continuously compares it with real-time sensor data.

⚡ The system detects:

* 🚨 Cyber Attacks
* 🌡️ Sensor Manipulation
* 📊 False Data Injection
* ⚠️ Anomalies
* 🔍 Operational Faults
